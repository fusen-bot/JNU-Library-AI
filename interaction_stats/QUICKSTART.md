# 快速开始指南

## 🚀 5分钟快速启动

### 第一步：启动分析工具

```bash
# 进入工具目录
cd interaction_stats

# 运行启动脚本
./run-analyzer.sh
```

**或者手动启动：**
```bash
python -m http.server 8081
```

### 第二步：打开浏览器

访问以下任一地址：
- **简化版（推荐）**: http://localhost:8081/simple-analyzer.html
- **完整版**: http://localhost:8081/index.html

### 第三步：上传数据文件

1. 点击页面上的 **"上传会话文件"** 按钮
2. 选择 `sessions/` 目录下的 `.jsonl` 文件
3. 可以一次选择多个文件

### 第四步：开始分析

上传完成后，工具会自动解析数据并显示：
- 📊 **数据总览**: 统计卡片和图表
- 📚 **图书交互**: 详细的交互数据表格  
- ⏰ **时间线**: 用户行为时间序列

## 📁 数据文件说明

### 支持的文件格式
- `.jsonl` - JSON Lines格式（推荐）
- `.json` - 标准JSON格式

### 示例数据结构
```json
{
  "session_id": "交互_01_20250928",
  "event_type": "book_clicked", 
  "timestamp": "2025-09-28T14:17:37.544Z",
  "book_title": "人工智能与未来教育",
  "book_author": "潘巧明",
  "book_isbn": "9787301339381"
}
```

## 🎯 主要功能

### 1. 多视图模式
- **单会话分析**: 深入分析特定会话
- **多会话汇总**: 整体趋势分析

### 2. 丰富的可视化
- 事件类型分布饼图
- 热门图书交互柱状图
- 用户行为时间线

### 3. 详细统计
- 总事件数、图书数、点击数
- 会话时长、平均悬停时间
- 图书交互详情表格

## ⚠️ 常见问题

### Q: 上传文件后没有数据显示？
**A**: 检查以下几点：
- 文件格式是否为JSONL
- JSON格式是否正确
- 文件是否包含必要的字段

### Q: 图表显示异常？
**A**: 尝试以下解决方案：
- 刷新页面重新加载
- 检查浏览器控制台错误信息
- 使用简化版工具

### Q: 服务器启动失败？
**A**: 确认以下环境：
- Python 3.6+ 已安装
- 端口8081未被占用
- 在正确的目录下运行脚本

## 💡 使用技巧

### 批量分析
1. 将多个会话文件同时选择上传
2. 切换到"多会话汇总"模式
3. 查看整体数据趋势

### 深度分析
1. 选择特定会话进行分析
2. 查看详细的时间线数据
3. 结合图书交互表格分析用户行为

### 数据导出
- 图表支持右键保存
- 表格数据可以手动复制
- 建议截图保存分析结果

## 🔧 高级配置

### 自定义端口
```bash
./run-analyzer.sh -p 9000
```

### 仅构建不启动
```bash
./run-analyzer.sh --build-only
```

### 不自动打开浏览器
```bash
./run-analyzer.sh -n
```
